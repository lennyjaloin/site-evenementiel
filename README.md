# Site Evenementiel

Projet separe en trois blocs :

- `backend_complete` : API Express/MySQL
- `frontend_complete/web` : frontend web React + Vite
- `frontend_complete/mobile` : frontend mobile React Native + Expo Go

## Commandes utiles

```bash
npm run dev:backend
npm run dev:web
npm run dev:mobile
```

## Web

Le frontend web a sa base dans `frontend_complete/web`.

## Mobile

L'application mobile est maintenant une vraie app React Native compatible Expo Go, en Expo SDK 54, avec :

- authentification
- navigation mobile native
- liste et detail des evenements
- reservation depuis mobile
- profil utilisateur
- ecran admin pour les comptes admin

Consulte `frontend_complete/mobile/README.md` pour le lancement et la configuration reseau Expo.
